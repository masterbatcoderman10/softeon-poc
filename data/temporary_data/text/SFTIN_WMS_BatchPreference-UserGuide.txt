4 
Batch Preference 
1. Overview 
You can use the Batch Preference screen for defining a batch preference type for creating the batch 
routes to perform batch picking. 
Menu Access: Masters > Outbound > Planning > Wave > Batch > Batch Preference 
2. Pre-requisites 
• 
Preference Type must be created. 
3. Defining preference types for creating the Batch 
The system displays the Cust. Bus Type as ‘D – Direct to Store’, by default. However, you can the 
Cust. Bus. Type, if required. 
Mandatory Values 
1) Specify/select the Preference Type to set the type of preference for route. 
2) Specify the Preference ID applicable for the defined preference type. 
3) Specify the Pref Seq # (Preference Sequence Number) by which batch preference will be sorted. 
4) Specify/select the Source Batch Pick Route and Target Batch Pick Route. 
5) Specify/select the Pick Type. 
Optional Values 
6) Select the Cust. Bus Sub Type from the list. 
7) Select the Quantity check box and specify the following details:  
• 
Select the Quantity Type as ‘Percentage’ or ‘Value’ to specify the order quantity range in 
terms of percentage or value. 
• 
Specify the order quantity Range values in from text boxes in Percentage or Value. 
• 
Select the unit of measurement for the order quantity (units, cases, or pallets) from the UOM 
list.
